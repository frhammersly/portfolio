A. Files Structure - top

Every file (HTML, CSS, JS) that was created by us has clear code, wrapped with comments - all this for your convenience.

  • css - CSS styles folder
  • fonts - webfont icons folder
  • img - folder for images
  • js - JavaScript files folder
  • php - PHP contact script folder
  • index.html - main HTML file

1. HTML Files and Structure

This Template has a Responsive layout and is based on the Bootstrap Framework. For more information about this visit Bootstrap site.

The default Bootstrap grid system utilizes 12 columns, making for a 940px wide container without responsive features enabled. With the responsive CSS file added, the grid adapts to be 724px and 1170px wide depending on your viewport. Below 767px viewports, the columns become fluid and stack vertically.

index.html is the main template file that contains the HTML structure of the template. Here is the general structure (shortened):

					
<!DOCTYPE html>
<html>
<head>
	<!-- Included CSS Files -->
</head>
<body>

</body>
</html>

C. Customization - top

You can simply change template's, make the following changes.

1. Replace Navigation Title

To change the Navigation Title replace the Invers text located in the template's index.html file with yours (line 52).


<a class="navbar-brand" href="index.html"><i class="fa fa-caret-up"></i> Invers</a>

2. Edit Text

All static texts can be modified within the index.html file. For example, to change the text "Jane Doe" open index.html and edit text on line 72.

3. Change Skill bars

To change the Skills percent of your site open index.html and replace your own skills percent.


<div class="col-md-4 skills">
   <h6 class="progress-title">DESIGNING</h6>
   <div data-progress="80" class="skills-item wow">
      <div class="skills-item-progress">
         <span class="counter">
                        <span class="count">80</span> %
         </span>
      </div>
   </div>
   <h6 class="progress-title">DEVELOPMENT</h6>
   <div data-progress="90" class="skills-item wow">
      <div class="skills-item-progress">
         <span class="counter">
                        <span class="count">90</span> %
         </span>
      </div>
   </div>

4. Change InView Animations

You can define a simple animation for any element which will be fired when element appears on screen. List of all animations can be found here.

To define an InView animation, add a wow fadeIn attribute with desired animation's name to any element. For example:


<div class="service wow fadeInUp">

5. Change the Email Address for Contact Form

Make sure that you change the default email address yourname@yourdomain.com to your own in contact_me.php file (line 19) within the php folder to receive emails when the contact form is submitted.


// Create the email and send the message
$to = 'yourname@yourdomain.com'; // Add your email address inbetween the '' replacing yourname@yourdomain.com - This is where the form will send a message to.
$email_subject = "Website Contact Form:  $name";
$email_body = "You have received a new message from your website contact form.\n\n"."Here are the details:\n\nName: $name\n\nEmail: $email_address\n\nPhone: $phone\n\nMessage:\n$message";
$headers = "From: noreply@yourdomain.com\n"; // This is the email address the generated message will be from. We recommend using something like noreply@yourdomain.com.
$headers .= "Reply-To: $email_address";	
mail($to,$email_subject,$email_body,$headers);
return true;			
?>

6. Add Links to Social Network Profiles

To add links to your social network profiles open index.html file and edit lines 549-552. Here is a default example:


<ul class="footer-social">
 <li><a href="#"><i class="fa fa-behance wow fadeInUp" data-wow-duration="1s" data-wow-delay="0.2s"></i></a></li>
 <li><a href="#"><i class="fa fa-google-plus wow fadeInUp" data-wow-duration="1s" data-wow-delay="0.4s"></i></a></li>
 <li><a href="#"><i class="fa fa-dribbble wow fadeInUp" data-wow-duration="1s" data-wow-delay="0.6s"></i></a></li>
 <li><a href="#"><i class="fa fa-facebook wow fadeInUp" data-wow-duration="1s" data-wow-delay="0.8s"></i></a></li>
</ul>

All icons can be replaced with new ones, their number can be increased/decreased. All available icons you can see on official website - Font Awesome.

To change icons, simply change the class names, for example: fa-facebook and fa-twitter. URLs can be attached to href attribute.
